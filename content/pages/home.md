Average Things Tech is a beginner-friendly place for Linux, open-source tools, Neovim, practical computing, and respectful technology advocacy.

This site is for normal people who want to understand technology without getting buried under jargon, hype, or unnecessary complexity.

## Start here

If you are new, begin with the [Start Here]({filename}start-here.md) page.

If you are moving from Windows to Linux, visit the [Guides]({filename}guides.md) section.

If you want practical posts, check [Articles]({filename}articles.md).

If you want respectful public technology requests, visit [Open Letters]({filename}open-letters.md).

## What this site covers

- Linux migration for beginners
- Vim and Neovim basics
- Terminal and command-line tools
- Git and GitHub
- Home servers and backups
- Open-source software
- Practical technology decisions
- Open Letters to technology companies

## The philosophy

First make it work.  
Then make it automatic.  
Then make it beautiful.  
Then make it public.

Boring first. Fancy later. Drama never.
