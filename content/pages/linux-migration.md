# Linux Migration Guide

This is the main guide for normal people moving from Windows to Linux.

The goal is not to become a Linux guru overnight.

The goal is to understand the basics, avoid unnecessary confusion, and slowly build confidence.

## Start here

- Watch Episode 1: Can a Normal Person Switch to Linux in 2026?
- Understand what Linux is
- Understand what changes from Windows
- Choose a beginner-friendly Linux distribution
- Learn the first basic terminal commands

## Philosophy

Simple tech. Real use. No drama.

We move slowly, carefully, and practically.
